# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in the FuseIQ open-source community a harassment-free experience for everyone.

## Standards

**Encouraged:** respectful feedback, value-first help, crediting upstream, responsible disclosure

**Unacceptable:** harassment, hate speech, doxxing, spam, sharing others' API keys or workspace data

## Enforcement

Report incidents to **hello@fuseiq.io** or moderators in [Discord](https://discord.gg/pDFgmqkf).

## Scope

Applies in GitHub issues/PRs, Discord, and community events for FuseIQ OSS repos.
